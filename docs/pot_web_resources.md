## Path of Titans web resources (reference)

These URLs are treated as primary external references for PALEO. They are not scraped or stored wholesale; instead we:

- Link to them from config/notes fields.
- Use them manually (or via future RAG tooling) when refining policies and fast-facts.

### 1. Official wiki (base game, mechanics, dinos)

- Home: `https://pathoftitans.wiki`
- Examples:
  - Dinosaur index pages under Herbivores/Carnivores (e.g., Allosaurus, Styracosaurus, Achillobator, Tylosaurus).
  - Game mechanics pages (buffs, debuffs, growth, communication, abilities).

### 2. Fandom wiki (supplemental, stats & mechanics)

- Main: `https://path-of-titans.fandom.com/wiki/Path_of_Titans_Wiki`
- Useful categories:
  - Dinosaur stats: `https://path-of-titans.fandom.com/wiki/Dinosaur_Stats`
  - Game mechanics: `https://path-of-titans.fandom.com/wiki/Category:Game_Mechanics`
  - Dinosaurs: `https://path-of-titans.fandom.com/wiki/Category:Dinosaurs`

These pages often provide more verbose descriptions, player tips, and aggregate stats that complement the official wiki.

### 3. GameServersHub curve overrides (up-to-date numeric stats)

- KTO index: `https://guides.gsh-servers.com/path-of-titans/guides/curve-overrides/modded-dinosaurs/kto/`
- Example creature pages:
  - KTOPachyrhino: `https://guides.gsh-servers.com/path-of-titans/guides/curve-overrides/modded-dinosaurs/kto/path-of-titans-ktopachyrhino`
  - KTOYang: `https://guides.gsh-servers.com/path-of-titans/guides/curve-overrides/modded-dinosaurs/kto/path-of-titans-ktoyang`

These pages contain numeric curve data (health, stamina, damage, multipliers) for modded creatures, which we reference in Instinct policy JSON notes and when tuning fast-facts.
