PALEO Technical Dashboard

PALEO is an AI system for dinosaur-themed survival gameplay. It combines computer vision, behavior modeling, and per-dinosaur Letta agents to produce instinct-driven actions and interpretable thought logs.

Current phase: foundation build and check-in documentation complete; model training and game integration are next.

Technical Direction

  • Game context: survival-style species interactions inspired by Path of Titans.
  • Data direction: Snapshot Serengeti + Kaggle predator/prey image sources.
  • Baseline: OpenCV rule policy for transparent first benchmark.
  • Main model: ResNet-18 transfer learning for behavior state prediction.
  • Agent orchestration: one Letta agent per dinosaur individual.

Reproducible Commands

  • python scripts/run_pipeline.py
  • python -m unittest discover -s tests -p "test_*.py"

Training and evaluation commands will be added once the behavior classifier pipeline is implemented.
