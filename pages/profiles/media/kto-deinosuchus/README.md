# KTO Deinosuchus media (optional)

Drop **creature-specific** screenshots or skin previews here and reference them from `data/kto-deinosuchus.json`:

- `screenshots/` — in-game captures for the **Gallery** section (recommended: PNG/WebP).
- `skins/` — one image per skin filename matching skin `id` (e.g. `coastal.webp`).

Use paths like `media/kto-deinosuchus/screenshots/01.webp` (relative to `pages/profiles/`).

Until these exist, the site uses Path of Titans Steam store screenshots as **generic in-game** placeholders and cycles them for skin cards.
