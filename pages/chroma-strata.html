Chroma Strata

Stack a base (flat color or full image variants), then transparent overlays per region. Add multiple files per layer as color options; Randomize picks one variant per layer. Press R anytime. Everything stays in your browser — nothing is uploaded.

Layers

If you add base images, randomize will pick among them; otherwise the swatch above shows through (overlays stack on top). Each file pick adds to this layer’s bank (use Clear to reset).

Tip: keep every file the same pixel size for a tight composite; mixed sizes are centered with contain.
